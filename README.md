# stress-tools

Stress testing toolkit Discord bot. Wraps MHDDoS (57 HTTP attack methods), slowloris, and HTTP flood tools for authorized load/stress testing. Requires authorization before use.

## Topics
python, discord, bot, stress-testing, ddos, pentest, security

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/stress-tools.git
cd stress-tools

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 stress-tools.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
